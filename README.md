# 災害リスク分析ツール

気象庁の警報情報と過去の気象データをもとに、都道府県ごとの災害リスクを可視化するWebアプリです。

---

## デモ

https://disaster-risk-analyzer.streamlit.app/

※ 一定時間アクセスがないとスリープします。
スリープ中は画面に「This app has gone to sleep」と表示されます。
画面内の「Yes, get this app back up!」ボタンを押すと1分ほどで復帰します。
## 制作背景

近年、日本各地で豪雨や猛暑などの気象災害が増加しています。「どの地域が今どれくらい危険なのか」を一目で把握できるツールがあれば、早めの行動につながると考え制作しました。

---

## 機能

- **警報情報の取得**：気象庁（JMA）のAPIからリアルタイムの警報情報を取得
- **気象データの取得**：Open-Meteo APIから過去8日分の最高気温・降水量を取得
- **リスクスコアの算出**：気温・降水量・警報の3指標をもとにスコアを計算
- **リスクランキング**：全都道府県のリスクを一覧で表示
- **地域トレンドグラフ**：選択した都道府県の直近リスク推移を折れ線グラフで表示
- **地図表示**：注意以上の地域をマップ上に色付きで表示

---

## リスクスコアの計算方法

| 指標 | 条件 | スコア |
|------|------|--------|
| 気温 | 35℃以上 | +3 |
| 気温 | 30℃以上35℃未満 | +2 |
| 降水量 | 50mm以上 | +3 |
| 降水量 | 20mm以上50mm未満 | +2 |
| 警報 | 発令中 | +3 |

- 合計7以上：**危険**
- 合計3以上：**注意**
- 合計3未満：**安全**

## 実行方法

```bash
# 依存ライブラリのインストール
pip install streamlit pandas requests folium streamlit-folium altair

# アプリの起動
streamlit run app.py
```

起動後、ブラウザで `http://localhost:8501` にアクセスしてください。  
画面上の「最新データを取得」ボタンを押すとデータが取得・保存されます。

---

## 工夫した点

- リスクの状況説明をデータと連動させ、固定テキストではなく実際のスコアに基づいたコメントを表示するようにしました
- 警報が1件もない場合は「現在、発令中の警報はありません」と明示し、ユーザーが安心できる表示にしました
- 地図は「注意」以上の地域のみ表示し、警報数に応じて円の大きさを変えることで、重要な情報が一目でわかるよう工夫しました
