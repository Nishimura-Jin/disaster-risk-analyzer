# 災害リスク分析ツール

気象庁の警報情報と気象データをもとに、都道府県ごとの災害リスクをリアルタイムで可視化・分析するWebアプリです。
過去8日間の実績データと今後7日間の予測データを組み合わせることで、現在の状況把握と今後の備えの両方に活用できます。

## 制作背景

近年、日本各地で豪雨や猛暑などの気象災害が増加しています。「どの地域が今どれくらい危険なのか」を一目で把握できること、そして「これからどうなるか」を事前に確認できることで、早めの行動につながると考え制作しました。

## 機能

### 概要タブ
- **最大リスク地域の表示**：全国で最もリスクの高い都道府県とその要因を表示
- **全体分布サマリー**：危険・注意・安全の県数をメトリクスで表示
- **ドーナツグラフ**：全国の危険度分布を割合で可視化
- **地方ブロック別リスク分布**：北海道・東北・関東など9ブロック単位で集計した横棒グラフ

### 警報タブ
- **警報情報の取得**：気象庁（JMA）のAPIからリアルタイムの警報・注意報情報を取得
- **警報種別の集計グラフ**：大雨・暴風・大雪など種別ごとの発令件数を横棒グラフで表示

### 地域分析タブ
- **リスクランキング**：全都道府県のリスクスコア・最高気温・降水量を一覧で表示
- **地方ブロック×危険度の複合フィルタ**：地方と危険度を組み合わせて絞り込み可能
- **地図表示**：注意以上の地域をマップ上に色付きで表示（円の大きさは警報数に対応）

### 気象予測タブ
- **過去8日間の実績**：選択した都道府県の気温・降水量の推移を表示
- **今後7日間の予測**：Open-Meteo予報APIから取得した予測値を同一グラフ上に重ねて表示
- **しきい値ライン**：注意・危険の基準ラインをグラフ上に表示し、リスクの高まりを視覚化

## リスクスコアの計算方法

気象庁の警報に依存せず、気温と降水量の実データから独自にスコアを算出しています。

| 指標 | 条件 | スコア |
|------|------|--------|
| 気温 | 35℃以上 | +3 |
| 気温 | 30℃以上35℃未満 | +2 |
| 降水量 | 50mm以上 | +3 |
| 降水量 | 20mm以上50mm未満 | +2 |
| 複合リスク | 30℃以上かつ20mm以上 | +2 |

- 合計6以上：**危険**
- 合計3以上：**注意**
- 合計3未満：**安全**

複合リスクは「高温かつ大雨」の状況で熱中症・土砂災害のリスクが重複することを考慮して加算しています。

## 実行方法

```bash
# 依存ライブラリのインストール
pip install streamlit pandas requests folium streamlit-folium altair

# アプリの起動
streamlit run app.py
```

起動後、ブラウザで `http://localhost:8501` にアクセスしてください。
初回起動時はデータを自動取得します。2回目以降は「最新データを取得」ボタンで更新できます。

## 工夫した点

- 気象庁の警報をスコアに含めず、気温と降水量の実データのみでリスクを算出しています。警報が出る前の段階でもリスクの上昇を検知できる設計です
- 「高温かつ大雨」の複合条件をスコアに加算しており、単一指標では捉えにくい複合的な危険状況を評価できます
- 過去の実績データと今後の予測データを同一グラフ上に表示することで、現在地と今後の見通しを一画面で確認できます
- 地方ブロック単位での集計を追加し、都道府県単位では見えにくい広域的な傾向を把握できるようにしました
